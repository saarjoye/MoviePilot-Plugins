# Panlink115

`Panlink115` 是一个 `MoviePilot v2` 插件骨架，目标是：

- 手动搜索盘链影视资源
- 优先展示 `115` 网盘链接
- 预留“加入 115”接口，为后续自动转存做适配层

## 当前能力

- 插件配置页支持填写盘链账号、密码、请求超时、搜索结果数量
- 插件详情页支持：
  - 关键字搜索
  - 展示搜索结果
  - 加载盘链资源详情
  - 仅展示 `115` 或展示全部网盘资源
  - 将 `115` 链接加入待转存队列

## 当前限制

- `加入 115` 目前只是占位接口，不会真的转存到你的 `115`
- 后续需要单独补 `115` 分享保存/转存逻辑

## 放入 MoviePilot 插件仓库时的结构

```text
plugins.v2/
  panlink115/
    __init__.py
    client.py
    requirements.txt
```

同时把根目录的 `package.v2.json` 中 `Panlink115` 条目合并到你的插件仓库根部 `package.v2.json`。
